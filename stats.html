Doodle

First things first, I have an idea and sketch it out on paper that I then show to other people to make sure I'm not crazier than I thought.

Design

I either design in Photoshop, Illustrator, or straight in the editor depending on the scenario and my familiarity with the style.

CLI

I use gulp for compilation of my SASS, minification/validation of my js, and inlining my svg sprites. I use browerSync to set up live reloading of my page during developlment.

Styles

I write my CSS in SASS (SCSS) and compile it using libSASS and autoprefixer. If the project won't be able to support flexbox, I use Susy for SASS to manage my grid.

Interactivity

If the project is complex enough to need a javascript framework, I generally lean towards AngularJS, but I'm looking for a reason to learn React and Ember.

Backend

I currently use Laravel for my backend on projects, but I am hoping to try the MEAN stack for my next big project.
