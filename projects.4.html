Goal: Create an easily customizable, dependency free module that visualizes css grids.
Challenge: This was my first time building a vanilla javascript module from scratch, so there was a lot to learn!

More and more, I find myself designing with code, and I don't intend to stop. It's an amazing feeling when you're able to craft a desisn by typing up some html and css, especially when you can do it quickly. Some things, however, are lost in this design process. One of which is visual guides, grids, and layout tools that you find in design & prototyping applications such as photoshop and illustrator.

Ghost Grid was created as a way to visualize complex css grid systems for designers who work in the browser.

There have been other attempts at bridging this gap which I have taken heavy inspiration from. Most notably is the Golden Grid System and its 'Golden Gridlet' overlay. My goal with Ghost Grid was to simply create an easily customizable, dependency free version of what Joni Korpi did with GGS that had a few more features.
